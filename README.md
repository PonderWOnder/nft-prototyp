# nft-prototyp

NFT Prototyp

## Install node.js

```
sudo apt install nodejs
```

## install truffle via npm

```
npm install -g truffle
```

## Start ganache

```
docker-compose up
```

## Exec into ganache container

```
docker exec -it nft-prototyp_node_1 /bin/sh
```

## To test run within the /nft-prototyp

```
truffle test
```
